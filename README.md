## 🚀 快速开始

### 1️⃣ 打开应用

- 用浏览器打开 `index.html`

### 2️⃣ 创建笔记

1. 点击左上角 `+新建` 按钮
2. 输入标题
3. 编写内容
4. **自动保存**（不需要手动点击）

### 3️⃣ 查看保存状态

- 右上角绿色脉冲 = 已保存
- 右上角橙色 = 保存中
- **状态实时更新**

### 4️⃣ 导出笔记

**导出为 TXT（最常用）：**

- 点击 `📄导出TXT`
- 用 Windows 记事本打开即可查看

**导出为 JSON（备份用）：**

- 点击 `📊导出JSON`
- 保存完整备份

### 5️⃣ 导入笔记

1. 点击 `📂导入`
2. 选择 JSON 文件
3. 笔记自动合并

## 📖 详细使用教程

### 文本格式化

**粗体：**

```
1. 选中文字
2. 点击工具栏 B 按钮
3. 文字变成 **粗体文字**
```

**斜体：**

```
1. 选中文字
2. 点击 I 按钮
3. 文字变成 *斜体文字*
```

**标题：**

```
1. 在行首输入内容
2. 点击 H1 按钮
3. 变成 # 标题文字
```

**代码块：**

```
1. 选中文字
2. 点击 </> 按钮
3. 文字变成 `代码`
```

**引用：**

```
1. 在行首输入内容
2. 点击 " 按钮
3. 变成 > 引用文字
```

### 标签使用

```
底部输入框 → 输入标签 → 用逗号分隔 → 按Enter键
```

示例：`重要, 工作, 2024年`

### 背景色选择

- 点击底部右侧的颜色圆块
- 选中色块会高亮显示
- 笔记背景颜色实时更新

### 搜索笔记

- 在顶部搜索框输入关键词
- 自动搜索标题、内容、标签
- 实时显示搜索结果

## 💾 数据保存说明

### 自动保存工作流程

```
修改笔记 → 状态显示"保存中" (橙色)
        → 3秒后自动保存
        → 状态变为"已保存" (绿色脉冲)
```

### 存储位置

- 笔记保存在浏览器本地存储
- 不同浏览器数据互不共享
- 清除浏览器数据会丢失笔记

### 存储容量

- 通常为 5-10MB
- 足够存储数千条笔记

### 备份建议

- 定期导出 JSON 文件保存
- 重要笔记单独导出 TXT
- 浏览器升级前做好备份

## ❓ 常见问题

### Q: TXT 导出的文件用记事本打开显示乱码？

**A:** 用以下方法：

1. 记事本 → 文件 → 打开 (Ctrl+O)
2. 文件类型改为"所有文件"
3. 找到导出的 TXT 文件
4. 编码选择 **UTF-8**
5. 打开

### Q: 笔记会丢失吗？

**A:**

- 笔记保存在浏览器本地
- 正常关闭浏览器不会丢失
- **但清除浏览器缓存会丢失**
- 定期导出 JSON 作为备份

### Q: 能否同步到多个设备？

**A:** 当前版本本地存储，可以：

- 导出 JSON 到云盘（OneDrive、百度网盘）
- 其他设备上导入该文件

### Q: 支持 Markdown 吗？

**A:** 支持输入 Markdown 语法，但不支持预览渲染

### Q: 如何恢复误删的笔记？

**A:**

- 删除无法恢复
- 如有备份可导入恢复
- 未来版本会加回收站
