💾 推荐使用流程
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

日常使用：
  1. 点击 +新建 创建笔记
  2. 输入标题和内容
  3. 内容自动保存（观察右上角状态）
  4. 用标签分类（如：工作、个人、学习）
  5. 用背景色标记重要笔记

导出备份：
  1. 每周导出一次 JSON 文件
  2. 重要笔记导出 TXT 文件
  3. 存放到云盘或本地备份
  4. 浏览器升级前务必备份

恢复数据：
  1. 点击 📂导入 按钮
  2. 选择之前导出的 JSON 文件
  3. 笔记自动合并恢复

【核心操作】
  新建笔记       → 点击 +新建
  编辑笔记       → 在编辑区输入
  删除笔记       → 点击 🗑删除
  搜索笔记       → 在搜索框输入

【格式化】
  粗体           → 选文本 → 点B
  斜体           → 选文本 → 点I
  下划线         → 选文本 → 点U
  代码           → 选文本 → 点</>
  引用           → 行首 → 点"
  标题           → 行首 → 点H1 (新增)

【功能按钮】
  保存状态       → 右上角 (绿色 = 已保存)
  字数统计       → 点📊 (新增)
  清除格式       → 点Aa
  添加标签       → 底部输入框
  选择颜色       → 底部颜色块

【导出导入】
  导出TXT        → 点📄导出TXT (可记事本打开)
  导出JSON       → 点📊导出JSON (用于备份)
  导入笔记       → 点📂导入 (选JSON文件)

Q: 如何打开应用？
A: 双击 index.html，或用浏览器打开

Q: 笔记会丢失吗？
A: 笔记保存在浏览器本地，不会因关闭而丢失
   但清除缓存会丢失，建议定期备份

Q: TXT导出显示乱码怎么办？
A: 用记事本"打开方式"选择编码为UTF-8

Q: 能否同步到其他设备？
A: 导出JSON到云盘，其他设备导入即可

Q: 为什么自动保存没有保存按钮？
A: 这是设计，自动保存更便利，无需手动

Q: 能否分类管理笔记？
A: 用标签分类，后续会加文件夹功能

Q: 最多能存多少笔记？
A: 5-10MB容量，足够存储数千条笔记
